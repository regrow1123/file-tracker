# AGENTS.md - 파일변경추적 프로젝트

## 역할
파일 변경/삭제 추적 시스템 개발 전용 에이전트.
대규모 파일시스템(수PB, 수백억 파일) 환경에서 파일시스템 벤더에 관계없이 동작하는 추적 솔루션 개발.

## 기술 스택
- auditd (커널 감사)
- eBPF / bpftrace / BCC (커널 레벨 추적)
- Falco (eBPF 기반 보안 모니터링)
- 중앙 로그 수집 (rsyslog, Elasticsearch, Loki)

## 제약
- 파일시스템 벤더 무관 (VFS 레벨)
- 서버 자원 최소 사용
- 변경/삭제된 파일 경로 + 누가(uid) 추적
